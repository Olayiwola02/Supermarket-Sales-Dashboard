# Supermarket-Sales-Dashboard
Interactive Power BI dashboard with sales, profit, and customer insights

An **interactive Power BI dashboard** analyzing **322.97K in sales**, **15.38K profit**, and customer behavior across **3 branches** (Giza, Alex, Cairo).


# Key Metrics
**Total Sales** |**322.97K**
**Total Profit** | **15.38K** 
**Units Sold** | **5,510** 
**Avg. Rating** | **6.97 / 10**

## Dashboard Pages

### Page 1 – **Executive Overview**

**KPIs**: Sales, Profit, Quantity, Rating  
**Breakdowns**: Branch, Customer Type, Product Line, Gender, Payment  
**Interactive slicers**: Branch · Month  

### Page 2 – **Insights & Detail**

**Product-line table**: Profit, Sales, Quantity, Rating  
**Monthly profit trends** (Jan–Mar)
